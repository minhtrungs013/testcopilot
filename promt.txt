Prompt 3 — Tenant middleware (rất quan trọng)
Viết

Create tenant middleware for multi-tenant SaaS ordering platform.

Requirements:

Incoming URL example:
domain.com/api/milk-tea-house/menu

Steps:

read tenant slug from params

find tenant by slug

verify tenant status active

inject tenant_id into req.tenant

If tenant not found:
return 404

If tenant suspended:
return 403

Use async middleware pattern.
